# HandModel Plugin para QGIS

## 📖 Descrição
O **HandModel** é um plugin para QGIS que permite importar, manipular e visualizar modelos manuais (2D/3D) diretamente no ambiente SIG.  
Ele foi desenvolvido para facilitar análises espaciais e apresentações interativas.

---

## 🚀 Instalação
1. Baixe o arquivo `.zip` do plugin.
2. No QGIS, vá em **Complementos > Gerenciar e Instalar Complementos...**
3. Clique em **Instalar a partir de ZIP** e selecione o arquivo baixado.
4. Após a instalação, o plugin estará disponível no menu de complementos.

---

## 🛠️ Funcionalidades
- Importação de modelos em formatos comuns (ex.: OBJ, STL, GeoJSON).
- Ajuste de escala, rotação e posição dos modelos.
- Visualização integrada em camadas 2D e 3D do QGIS.
- Exportação dos modelos ajustados para novos arquivos.
