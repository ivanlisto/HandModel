about_pt=HandModel Plugin é uma extensão para o QGIS desenvolvida para facilitar análises hidrológicas e geomorfológicas a partir de modelos digitais de elevação (DEM). O plugin organiza os resultados em um workspace definido pelo usuário e adiciona automaticamente as camadas geradas ao projeto aberto no QGIS, já com simbologia aplicada. Principais funcionalidades: cálculo de declividade em graus ou percentual com simbologia automática em quartis; extração da rede de drenagem a partir de DEMs com algoritmos GRASS; integração com workspace para salvar resultados em GeoTIFF; interface amigável para seleção de parâmetros e acompanhamento do progresso; automação para carregar camadas diretamente no projeto QGIS.
